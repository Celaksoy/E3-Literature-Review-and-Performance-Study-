# NASA E³ Engine Analysis and Gasturb Simulation

## Project Description

This project is the final deliverable of the *UCK 470E – Jet Engine System Design with Practical Applications* course.

The study focuses on a comprehensive analysis of the NASA Energy Efficient Engine (E³) program, a major initiative from the late 1970s aimed at advancing aircraft engine efficiency and performance.

In addition to an in-depth literature review based on NASA technical reports, the project includes practical performance simulations using Gasturb software.

## Scope of the Study

### Literature Review

* Detailed analysis of the NASA E³ engine program
* Engine design characteristics
* Control systems
* Secondary Air System (SAS)
* Cost and risk analysis
* Performance improvements and expected benefits

###  Gasturb Simulation

Using engine parameters obtained from NASA reports, single cycle design point simulations were performed in Gasturb for:

* Take-off condition
* Cruise condition
* Maximum climb condition

The objective was to approximate NASA’s results (originally obtained using NPSS) and validate the thermodynamic cycle understanding.

### Benchmark Analysis

A comparative study was conducted between the E³ engine and other turbofan engines (both historical and modern) to evaluate its technological impact and contributions.

##  Tools Used

* Gasturb
* NASA Technical Reports

## How to Use

1. Open Gasturb software
2. Load the provided ".cym" files
3. Review simulation parameters and results
4. Refer to the report for detailed analysis and explanations

## Outcomes

* Understanding of turbofan engine thermodynamic cycles
* Approximation of NASA E³ engine performance
* Validation of simulation results through comparison
* Insight into historical engine development and design evolution

## Significance

The NASA E³ program represents a milestone in aircraft propulsion technology. This project highlights its innovations and demonstrates how modern tools like Gasturb can be used to analyze and replicate complex engine behavior.
